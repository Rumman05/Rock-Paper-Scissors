# Rock Paper Scissors Game

A simple, interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript. Play against the computer in a best-of-5 match with a clean interface.

## Features

- Interactive button-based gameplay
- First to 5 wins format
- Replay functionality
- Responsive design
- Smooth hover animations

## How to Play

1. Open `index.html` in your web browser
2. Click on Rock, Paper, or Scissors to make your choice
3. Watch the results display update with each round
4. First player to reach 5 wins is the champion
5. Click "Replay" to start a new game

## Game Rules

- Rock beats Scissors
- Paper beats Rock
- Scissors beats Paper
- Same choices result in a draw

## Files

- `index.html` - Game structure and layout
- `styles.css` - Styling and animations
- `script.js` - Game logic and interactivity

## Installation

1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Start playing immediately - no setup required!

## Browser Compatibility

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
